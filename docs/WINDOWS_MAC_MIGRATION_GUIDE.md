### **Control Panel / System Settings** ‚Üí **System Preferences**
**Windows:** Control Panel ‚Üí System  
**Mac:** System Preferences  
**Ubuntu:** Click "System Preferences" in the sidebar

Manage everything in one place:
- Display & brightness (like Windows Display settings)
- WiFi & Bluetooth (like Mac Control Center)
- Sound & notifications
- Battery & power settings
- Appearance & themes
- Do Not Disturb mode

### **File Explorer / Finder** ‚Üí **Files**
**Windows:** File Explorer  
**Mac:** Finder  
**Ubuntu:** Click "Files" in the sidebar

Familiar features:
- Grid and list view modes
- Right-click context menus
- Breadcrumb navigation
- File preview icons
- Quick search
- Drag & drop (coming soon)

### **Device Manager** ‚Üí **Device Manager**
**Windows:** Device Manager  
**Mac:** System Information  
**Ubuntu:** Click "Device Manager" in the sidebar

Manage all your hardware:
- Graphics cards (NVIDIA/AMD/Intel)
- Network adapters (WiFi/Ethernet)
- USB devices
- Printers & scanners
- Bluetooth devices
- Update drivers with one click

### **Microsoft Store / App Store** ‚Üí **App Store**
**Windows:** Microsoft Store  
**Mac:** App Store  
**Ubuntu:** Click "App Store" in the sidebar

Discover and install apps easily:
- Visual Studio Code
- Spotify
- Discord
- Steam
- Zoom
- Chrome/Firefox
- And thousands more!

### **Task Manager / Activity Monitor** ‚Üí **Performance**
**Windows:** Task Manager (Ctrl+Shift+Esc)  
**Mac:** Activity Monitor  
**Ubuntu:** Click "Performance" in the sidebar

One-click system optimization:
- Clean system cache
- Manage startup apps
- Free up disk space
- Update system
- Check system health score

### **Disk Cleanup / CleanMyMac** ‚Üí **Performance**
**Windows:** Disk Cleanup  
**Mac:** CleanMyMac  
**Ubuntu:** Click "Performance" ‚Üí "Clean System"

Remove junk files:
- System cache
- Browser data
- Old logs
- Temporary files
- Trash/Recycle Bin

## üé® Making Ubuntu Feel Like Home

### **For Windows Users**

**1. Taskbar at the Bottom**
```bash
# Ubuntu has a dock by default, but you can customize it
# System Preferences ‚Üí Appearance ‚Üí Dock Position
```

**2. Start Menu**
- Press `Super` (Windows key) to open Activities overview
- Or use the "App Store" to browse all applications

**3. Window Controls**
- Windows: X (close) is on the right ‚úÖ Same in Ubuntu!
- Minimize/Maximize buttons also on the right

**4. Copy/Paste**
- Same shortcuts! Ctrl+C, Ctrl+V, Ctrl+X ‚úÖ

**5. File Management**
- Your files are in `/home/username` (like C:\Users\username)
- Documents, Downloads, Pictures, Music folders work the same way

### **For Mac Users**

**1. Menu Bar**
- Ubuntu has a top panel like macOS ‚úÖ
- Date/time on the right
- System icons on the right

**2. Dock**
- Ubuntu has a dock on the left by default
- Customize in System Preferences ‚Üí Dock
- Can move to bottom like macOS

**3. Trackpad Gestures**
```bash
# Many macOS-like gestures work:
# - Two-finger scroll
# - Three-finger swipe between workspaces
# - Pinch to zoom
```

**4. Spotlight Search**
- Press `Super` (Command key) for Activities search
- Or use the search bar in any window

**5. Command Key Shortcuts**
- Ctrl key in Ubuntu = Command key in macOS
- So Ctrl+C = Command+C

### **Change Password**
1. Users (sidebar)
2. Click your user
3. "Change Password"
4. Type old and new password

### **Check Disk Space**
1. Files (sidebar)
2. Right-click any folder
3. "Properties" shows size
4. Or: Storage (sidebar) for full overview

## ‚ùì FAQ

### **Q: Will my files from Windows/Mac work?**
A: Yes! Documents, photos, videos, music all work the same way.

### **Q: Can I use Microsoft Office?**
A: Use LibreOffice (free, pre-installed) or Office Online in browser.

### **Q: What about my iPhone/Android?**
A: Android works great! iPhone has limited support but basics work.

### **Q: Is it safe to update?**
A: Yes! Updates are tested. Your system won't break.

### **Q: Can I dual-boot with Windows?**
A: Yes! Many users keep Windows for specific apps.

### **Q: What if I need help?**
A: This app has built-in help, plus Ubuntu community is very friendly!

### **Q: Do I need to use the terminal?**
A: Nope! This interface lets you do 99% of tasks without it.

### **Q: Is Ubuntu free?**
A: Yes! Completely free, forever. No subscriptions.

### **Q: Will my printer work?**
A: Most printers work automatically. Check Device Manager.

### **Q: Can I play games?**
A: Yes! Steam has thousands of Linux games + Proton for Windows games.
